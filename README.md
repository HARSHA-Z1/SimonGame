# Simon Game
This is a simple Simon Game implemented using HTML, CSS, JavaScript, and jQuery. The game challenges players to memorize sequences of colors and repeat them in the correct order.

## Live Demo
You can play the game [here](https://harsha-z1.github.io/SimonGame/).

## Features
- Classic Simon Game mechanics
- Responsive design for various screen sizes
- Visual and audio feedback for each button press

## Installation
To run this project locally, follow these steps:
`git clone https://github.com/harsha-z1/SimonGame.git`
`cd SimonGame`
Open `index.html` in your web browser to play the game.

## Usage and Contributing
Simply visit the live demo link above or run the game locally as described in the installation section. Click on the buttons in the order shown by the game to progress through the levels.
